Datoer

Et dato objekt kan lave ret enkelt

    const idag = new Date()

På dette objekt kan der kaldes forskellige indbyggede metoder til at få adgang til de forskellige elementer af dagens dato

    const aar = idag.getFullYear() 
    // Giver nuværende år (f.eks. 2023)
    const maaned = idag.getMonth() 
    // Giver den aktuelle måned (første måned er index 0, så januar er 0, februar er 1 osv.) 
    const dag = idag.getDate() 
    // giver den aktuelle dag i måneden (1-31)

Ønsker man at en måned skrives istedet for et tal, og på dansk kan det også gøres

    const paentFormateretDato = new Intl.DateTimeFormat('da-DK', {
        year: 'numeric',
        month: 'short',
        day: 'numeric'
    }).format(iDag);
