Loops i javascript

For loop

Et for loop der skal iterere et defineret antal, af en mængde elemter kan skrives: 

    for (let i = 0; i < 5; i++) {
        console.log(i) //vil udskrive 0,1,2,3,4
      }

For in

I et for in loop, iterers der igennem en hel liste af elementer som sådan


    for (variabel in array) {
        kode
      }

    const numbers = [1,2,3,4,5]
    for (number in numbers) {
        console.log(number) //vil vise 1,2,3,4,5       
      }
