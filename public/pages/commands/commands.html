Kommandoer

Git kommandoer

git clone (url)

I stedet for ( url ), indsættes et webadresse til et git repository. Når man kører kommandoen, skabes en mappe der hvor man kalder kommandoen fra, med det en kopi af det linkede repository.

git commit -m "besked"

Dette opretter et commit, med den besked som står indenfor ”” i stedet for besked.

git push

Dette sender dine commit op til dit repo.

Generelt

code .

Denne kommando åbner visual studio, med udgangspunkt i den mappe, hvorfra du kører kommandoen.

Ctrl + c

Dette afslutter den igangværende process, '+' skal forstås som at trykke control tasten og c tasten samtidigt.

cd ..

Går en mappe op

cd (mappe)

Går ind i en mappe

Pwd

Print working directory, eller hvor er jeg, giver det sted du kører kommandoen fra.

Mkdir (navn)

Laver en mappe det navn du vælger.

Node

node (filnavn)

Dette får node til at kører den fil der angives, dette kunne fx være 'node .\app.js', der så kører din app.js fil

Npm

Npm install

Hvis du modtager en fil med depencies, der er angivet i dens package.json, vil denne kommando installere de nødvendige dependencies. Kan også forkortes til ”npm i”

Npm init

Dette laver et nyt node.js projekt, hvor den opretter en package.json, ud fra nogle spørgsmål om author, licens, repo, keywords mfl.

Npm init –y

Dette bruges til at lave et nyt default node.js projekt, hvor der ikke skal besvares sprøgsmål om projektet, men laver en standart package.json fil.

Npm install express

Dette får packagemanageren til at installere dependencien express.
