At læse filer

For at kunne læse en fil, er det nødvendigt at importere node.js filsystem

Import fs from 'fs'

Når dette er importeret, kan man læse en fil

    
    export function readpage (filepath) {
        return fs.readFileSync(filepath).toString()
    }

Dette eksempel bruger fs til at læse en fil, her synkront, ud fra den sti der er givet og returnerer det som en String.

At læse fra en json fil

Her er et eksempel på at læse fra en jsonfil, som egentlig ikke adskiller sig fra at læse fx en html fil. 

    async function readMountains() {
        const mountains = await fs.readFile(`data/mountains.json`, "utf-8", (err) => {
          if (err) {
            console.log("failed to read file");
          }
        });
      
        const mountainsJson = JSON.parse(mountains);
        return mountainsJson;
      }

Det som her adskiller sig fra det første eksempel er brugen af async/await, og at det som metoden returnerer er i json format.
