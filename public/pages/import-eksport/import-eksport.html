Import af filer eller funktioner

eksport af funktion

I javascript kan man eksporterer funktioner ved at skrive export foran for funktionen 

    export function myfunction() {
        //noget kode
    }

Import

En "gammel" måde at importerer på, er ved bruge af 'require', der er en del af CommonJS-modulesystemet, når man bruger require, kan der indlæses funktioner, variabler eller objekter fra andre filer eller indbyggede Node.js-moduler. Brug af require kunne gøres sådan 

    const express = require('express')

Den nyere metode er 'import', der blev implementerer i ES6 (2015), alle import statements placeres først i koden, og henter modulerne asynkront. For at javascript filerne kan genkendes som moduler, skal der dog erklæres i package.js, at programmet gør brug af moduler ("Oprettelse af nyt projekt"). En import kunne se således ud: 

    import fs from 'fs'
