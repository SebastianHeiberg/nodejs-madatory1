Start project med Node.js and Express

  1. Lav en app.js fil
  2. Brug kommandoen ”npm init –y”

    3. - for forklaring af kommandoen se kommandoer

  3. Brug kommandoen “npm install express”

    - for forklaring af kommandoen se kommandoer

  4. Tilføj ”type”: ”module” til din package.json under dependencies, dette vil gøre at du kan bruge import / eksport i dine .js filer.
    5.  {
        "dependencies": {
            "express": "^4.18.2"
        },
        "name": "aflevering",
        "version": "1.0.0",
        "main": "app.js",
        "type": "module",
        "devDependencies": {},
        "scripts": {
            "test": "echo \"Error: no test specified\" && exit 1"
        },
        "keywords": [],
        "author": "",
        "license": "ISC",
        "description": ""
        }
  5. Import express og path, hvilket gør at man kan benytte sig af express og path, path skal bruges senere
    6.  
        import express from 'express'
        import path from 'path'
  6. Så skal der instantieres en ny express app
    7.         Const app = express()
  7. Vi laver en variable der hedder port, som fortæller hvilken port at appen lytter på
    8. Const port = 8080
  8. Så skal appen sættes til aktivt at lytte på porten, og give en besked, hvis det fungerer.
    9.         app.listen(PORT, () => {
        console.log("App is running on port: ", PORT)
    })
  9. Der kan tilføjes en håndtering af get request på localhost:8080/ , og give en html side tilbage sådan her. Path.resolve tager imod en relativ path, og returnerer en absoulut path
    10.             
        app.get("/", (req, res) => {
            res.sendFile(path.resolve("./public/home.html" ))
        })
  10. Brug kommandoen: ”node .\ app.js” for at køre programmet.
