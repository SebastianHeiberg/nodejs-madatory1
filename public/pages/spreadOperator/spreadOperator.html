Spread operatoren

Man kan bruge spreadoperatoren, hvis man fx vil kopierer et array ind et nyt, og ikke komme ud for, at hvis der ændres i det første, så påvirker det det nye. Et eksempel kunne være følgende:

    const oprindeligtObjekt = { a: 1, b: 2 }
    const nytObjekt = { ...oprindeligtObjekt }

Man kan også sætte kopierer elementer ind midt i et array 

    const kropsdele = ["skuldre", "knæ"];
    const børnesang = ["hoved", ...kropsdele, "og", "tå"];

Hvis man printede denne børnesang, ville det give et output med ["hoved", "skuldre", "knæ", "og", "tå"]
