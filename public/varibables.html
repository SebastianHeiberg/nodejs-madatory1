Variabler

Javascript er ikke typestrict, så når der erklæres en variabel, skal der ikke specificeres hvilken type data der brues, som i fx java. JavaScript vil automatisk bestemme datatypen baseret på værdien, som variablen tildeles.

Const

En const (konstant) kan benyttes, når man ønsker at ens variable ikke skifter værdi, datatype, eller datastruktur. Dog hvis det er en form for liste, kan der godt tilføjes eller fjernes elementer fra listen.

En const variable kan erklæres ved:

 const minVariabel

Den kan initaialiseres ved: 

 const minVariabel = 4

let

En let variabel kan bruges, hvis man forventer eller ønsker at variablen kan skifte værdi, datatype, eller struktur. Man kan derfor instantierer og tilskrive en let varibel, og efterfølgende ændre den som her: 

        let minvariabel = 1
        let minvariabel = 'Hello'
